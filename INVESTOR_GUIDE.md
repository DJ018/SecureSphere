## üìä API Endpoints

### Learning APIs
- `GET /api/learn/module/:moduleId` - Get module content
- `POST /api/learn/quiz` - Submit quiz answers
- `POST /api/learn/progress` - Track user progress
- `POST /api/learn/certificate` - Generate certificates

### Services APIs
- `GET /api/services` - Get services catalog
- `POST /api/services/request` - Submit service request
- `POST /api/services/calculate-price` - Calculate pricing

### Lab APIs
- `POST /api/sql` - SQL injection lab
- `POST /api/phish` - Phishing detection lab
- `POST /api/password` - Password strength checker
- `GET /api/network` - Network scan simulator
- `POST /api/iam` - IAM access control

- `GET /api/games/leaderboard` - Get leaderboard
- `POST /api/games/leaderboard` - Add to leaderboard
